# my-components

This is a project that uses the Stencil web component library
to define multiple web components as a way of teaching how to use Stencil.

The command "npm run coverage" runs all the spec and e2e tests,
and outputs a code coverage report.
Currently the coverage is not 100%.
All the tests pass, but they cannot pass without running the code
that the coverage report claims is not covered.
